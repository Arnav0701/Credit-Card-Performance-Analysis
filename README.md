# Credit-Card-Performance-Analysis

## 📌 Project Objective

To develop a comprehensive credit card weekly dashboard that provides real-time insights into key performance metrics and trends, enabling stakeholders to monitor and analyze credit card operations effectively.

---

## 🔄 Project Workflow

### 1️⃣ Data Collection

* Obtained raw credit card dataset in CSV format.
* Dataset includes transaction details and customer-level information.

### 2️⃣ Database Creation

* Created a MySQL database.
* Imported the CSV data into structured tables for better data management and integrity.

### 3️⃣ Data Connection

* Connected the MySQL database to Power BI.
* Established a live data connection for analysis and reporting.

### 4️⃣ Data Preprocessing

* Cleaned and transformed data using Power BI Transform Data:

  * Corrected data types
  * Handled null/blank values
  * Removed inconsistencies
* Ensured data model readiness for analysis.

### 5️⃣ Data Modeling

* Created calculated columns and DAX measures such as:

  * Total Customers
  * Delinquent Customers
  * Delinquency Percentage
  * Revenue-related metrics
* Built relationships between relevant tables where required.

### 6️⃣ Dashboard Development

Developed two interactive dashboards:

#### 📊 Credit Card Transactions Dashboard

* Weekly revenue tracking
* Transaction volume analysis
* Performance trends
* Key financial indicators
